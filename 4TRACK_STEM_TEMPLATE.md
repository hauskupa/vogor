# 4-Track Stem Template

Use this as the default planning template when bouncing songs for the 4-track album mixer.

The goal is not literal DAW export by instrument category.
The goal is to create 4 playable layers that feel good to mix on a 4-track style device.

## Default Roles

1. `Pulse`
2. `Weight`
3. `Frame`
4. `Air`

## Role Guide

### `Pulse`

What it does:
- Pushes the song forward
- Carries rhythm and motion
- Gives the user something immediate to lock onto

Typical content:
- Live drums
- Drum machine
- Percussion
- Rhythmic guitar chugs if needed

### `Weight`

What it does:
- Holds the bottom end
- Adds body and low-mid support
- Makes the song feel grounded

Typical content:
- Bass
- Low synth
- Low reinforcement layers

### `Frame`

What it does:
- Defines the song structure
- Makes the song recognisable
- Carries the core harmony or main riff language

Typical content:
- Basic guitars
- Piano
- Keys
- Main chord/riff material

### `Air`

What it does:
- Adds space, lift, mood and identity
- Changes the emotional feel when brought in
- Feels like the "color" layer

Typical content:
- Atmospheric guitars
- Delays and reverbs if printed on purpose
- Pads
- Melodic overdubs
- Hooks
- Vocals, if vocals behave more like texture than lead center

## Rules

- Every stem should make sense on its own.
- No single stem should contain "the whole song".
- `Pulse + Weight` should already feel musical.
- `Frame` should make the song identifiable.
- `Air` should add mood, not just clutter.
- The 4 stems should feel performable, not just technically separated.

## Vocal Rule

If a song has vocals, do not automatically create a dedicated vocal stem.

Ask:
- Is the vocal part of the main identity?
- Or is it more of a texture / floating layer?

If it is a main identity element:
- Put it in `Frame`

If it acts more like texture or atmosphere:
- Put it in `Air`

## DAW Workflow

Create 4 buses:
- `Pulse Bus`
- `Weight Bus`
- `Frame Bus`
- `Air Bus`

Route the arrangement into those buses intentionally.

Before exporting, test these combinations:
- `Pulse`
- `Pulse + Weight`
- `Weight + Frame`
- `Frame + Air`
- all 4 together

If the song falls apart in these combinations, rebalance the routing.

## Quick Checklist Per Song

- What is the `Pulse` element in this song?
- What gives the song `Weight`?
- What makes the song recognisable in `Frame`?
- What adds emotion, width or lift in `Air`?

## Working Principle

Think like this:

This is not:
- `drums`
- `bass`
- `guitar`
- `vox`

This is:
- 4 layers the listener can perform with.
